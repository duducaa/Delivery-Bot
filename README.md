# TO RUN
- create a virtual environment and activate it
```
python3 -m venv venv
source path/to/the/venv/bin/activate
```
- install the requirements
```
python3 -m pip install requirements.txt
```
- run the application
```
python3 file.py
```

# MULTIPLE RUN
```
./script.bash 50 10 # 50 times with 10ms of view update
```
